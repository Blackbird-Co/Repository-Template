### Repository-Template:

Template para repositórios de documentação de códigos, notebooks e scripts da Blackbir (Mantendo padrão de documentação)

Repositórios devem conter um arquivo README.md para descrição da utilização do projeto salvo no repositório.
Arquivo idealmente deve conter uma descrição de cada arquivo salvo no repositório e a função do projeto de maneira geral

(Ex.: 
exemplo.py:
Script principal para realização da automação

exemplo.ipynb:
Arquivo notebook com construção e análise exploratória
)

#### Deploy:

Em casos em que o script final seja um código utilizado em aplicação (feito deploy) é ideal que a plataforma onde
o mesmo está seja apontada e identificado o nome da aplicação do mesmo.

(Ex.: Deploy do script realizado na plataforma do microsoft azure - functions na aplicação "template-exemplo")
